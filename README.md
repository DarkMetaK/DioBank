<!-- GETTING STARTED -->
## Como rodar o projeto

É possível acessar a plataforma acessando o link: https://darkmetak-diobank.netlify.app/

1 - Clone o repositório

2 - Instale as dependências
```sh
npm install
```

3 - Execute o projeto
```sh
npm run dev
```
